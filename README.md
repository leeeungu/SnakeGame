# 🎮 SnakeGame
 2DSpriteGame

## 📌 Introduction
SDL library를 사용하여 UDP/TCP를 실습한
개발 2D 토이 프로젝트입니다.

## 📂 Project Structure
```text
📦 ProjectRoot
┣ 📂 SnakeGame    # 게임 소스 코드
┣ 📂 Resources      # 게임 에셋 (모델, 텍스처, 사운드 등)
┣ 📂 Include          # SDL 헤더 파일
┣ 📜 .gitignore      # Git 무시 파일 목록
┣ 📜 README.md       # 프로젝트 소개 문서
┗ 📜 SnakeGame.sln  # VS 프로젝트 파일
```

## 🛠️ Setup & Installation
Prerequisites
SDL2.0 library
Visual Studio / Rider (C++ 사용 시 필요)
Git
Build & Run
